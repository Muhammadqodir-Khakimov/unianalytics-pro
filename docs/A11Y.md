# Accessibility (a11y) — Web Content Accessibility Guidelines 2.1 AA

## Avtomatik testlar

### axe-core (Storybook bilan)
```bash
npm i -D @storybook/addon-a11y axe-core
```
`.storybook/main.ts` ga `@storybook/addon-a11y` allaqachon qo'shilgan.

### Lighthouse CI
```bash
npm i -g @lhci/cli
lhci autorun --collect.url=http://localhost:3000 --collect.url=http://localhost:3000/dashboard
```

### Playwright + axe
```bash
npm i -D @axe-core/playwright
```

`e2e/a11y.spec.ts`:
```ts
import { test, expect } from '@playwright/test';
import AxeBuilder from '@axe-core/playwright';

test('landing has no critical a11y violations', async ({ page }) => {
  await page.goto('/landing');
  const r = await new AxeBuilder({ page }).withTags(['wcag2a', 'wcag2aa']).analyze();
  expect(r.violations.filter(v => v.impact === 'critical')).toEqual([]);
});
```

## Checklist (manual review)

- [x] Barcha `<img>` da `alt` attribute (yoki `alt=""` dekorativ uchun)
- [x] Barcha buttonlarda ko'rinadigan text yoki `aria-label`
- [x] Form `<label>` har bir input bilan bog'langan
- [x] Color contrast 4.5:1+ (oddiy text), 3:1+ (yirik text)
- [x] Keyboard navigation — Tab, Shift+Tab, Enter, Escape
- [x] Focus indicator har bir interactive element'da ko'rinadi
- [x] Heading hierarchy (h1 → h2 → h3, sakrash yo'q)
- [x] ARIA roles to'g'ri ishlatilgan (Ant Design auto handles)
- [ ] Screen reader bilan test (NVDA, VoiceOver)
- [ ] Mobile touch target 44×44 px minimum

## Web Vitals targets

| Metrika | Yaxshi | Yaxshilash | Yomon |
|---------|--------|-----------|-------|
| LCP (Largest Contentful Paint) | ≤ 2.5s | 2.5–4s | > 4s |
| INP (Interaction to Next Paint) | ≤ 200ms | 200–500ms | > 500ms |
| CLS (Cumulative Layout Shift) | ≤ 0.1 | 0.1–0.25 | > 0.25 |
| FCP (First Contentful Paint) | ≤ 1.8s | 1.8–3s | > 3s |
| TTFB (Time to First Byte) | ≤ 800ms | 800–1800ms | > 1800ms |
