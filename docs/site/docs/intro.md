# UniAnalytics PRO

**OLAP-asoslangan talaba reytingi tahlili va Business Intelligence platforma** O'zbekiston oliy ta'lim muassasalari uchun.

## Asosiy imkoniyatlar

- OLAP Cube va Pivot tahlili (Slice / Dice / Drill-down / Roll-up)
- ML: drop-out bashorati (XGBoost+SHAP), klasterlash (K-Means/DBSCAN), anomaliya aniqlash
- 7 ta rol: SuperAdmin, UniversityAdmin, Rector, MinistryViewer, Dekan, Teacher, Student, Public
- HEMIS bilan avtomatik sinxronizatsiya
- Telegram bot, AI Tutor (Claude/OpenAI), PWA, React Native mobil app
- Multi-tenancy + billing (Click/Payme/Stripe)
